# Cactus Page 🌵

## 📌 Description  
This is a simple web page with a cactus image and styling using HTML and CSS.

## 🚀 Technologies Used

- CSS3

## 🔧 How to Run
1. Download the repo or clone it.
2. Open `index.html` in your browser.
